错误如下：

 /Users/scott/local/tomcat/apache-tomcat-7.0.64/bin/catalina.sh run
 [2016-04-05 11:58:09,377] Artifact MVNFHM:war exploded: Server is not connected. Deploy is not available.
 Connected to server
 [2016-04-05 11:58:09,798] Artifact MVNFHM:war exploded: Artifact is being  deployed, please wait...
 错误: 代理抛出异常错误: java.rmi.server.ExportException: Port already in use: 1099; nested exception is: 
 java.net.BindException: Address already in use
 Disconnected from server

在命令行输入

sudo lsof -i :1099

结果如下：

COMMAND  PID USER FD TYPE DEVICE SIZE/OFF NODE NAME
java  26499 scott  24u  IPv6 0xa1055320cb8c4e3  0t0  TCP *:rmiregistry (LISTEN)

然后根据PID杀进程：

sudo kill -9 26499

至此问题已顺利解决
